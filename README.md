# Retail Sales Analysis Dashboard

## Objectif
Analyse des ventes d'un commerce de détail afin d'identifier 
les tendances, les catégories les plus performantes et le profil des clients.

## Outils utilisés
- Power BI (dashboard interactif)
- Dataset : Retail Sales Dataset (Kaggle)

## Analyses réalisées
- Chiffre d'affaires total
- Ventes par catégorie de produit
- Évolution des ventes dans le temps
- Répartition des ventes par genre
